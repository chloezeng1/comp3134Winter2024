What is the definition of a server port?
A server port is a virtual point where network connections start and end.
 Ports are software-based and are used by the operating system to manage network resources.
 They allow computers to distinguish between different services (like web traffic or email) running on the same computer.
What is the server port range?
The range for TCP and UDP ports is 0 to 65535.
How many bits does the number above represent?
The port number range (0 to 65535) requires 16 bits for representation.
